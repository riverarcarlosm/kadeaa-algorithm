# Overview

Kadeaa Backend Developer test.

# Installation

Install dependencies:

```bash
$ npm install
```

#Usage

Print in console the numbers from 1 to 100:

```bash
$ npm start
```

Print a specific number series e.g.: 1 to 10:

```bash
$ node index.js 10
```

# Test

Run simultaneous tests and compare string ouputs:

```bash
$ npm test
```
